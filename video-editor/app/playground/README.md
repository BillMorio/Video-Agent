# Playground

This folder contains test pages for UI experiments and API testing.

## Available Test Pages

| Page | Path | Description |
|------|------|-------------|
| **Index** | `/playground` | Main hub with links to all test pages |
| **UI Components** | `/playground/ui` | Test UI components in isolation |
| **API Tests** | `/playground/api` | Test API endpoints and integrations |
| **FFmpeg** | `/playground/ffmpeg` | Test FFmpeg operations (trim, fit, zoom) |

## How to Add a New Test Page

1. Create a new folder under `app/playground/[your-test-name]/`
2. Add a `page.tsx` file inside
3. Update this README with the new page details
